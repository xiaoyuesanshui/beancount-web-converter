# Beancount 账单转换器 - 使用说明

## 功能特性

- ✅ 支持支付宝账单（CSV/XLSX）
- ✅ 支持微信账单（CSV/XLSX）
- ✅ 自动识别账单类型
- ✅ 自动切换配置模板
- ✅ 支持自定义规则
- ✅ 支持任意年份的账单

## 使用步骤

### 1. 上传账单文件

支持的文件格式：
- 支付宝：CSV、XLSX
- 微信：CSV、XLSX

上传后系统会自动识别账单类型（微信/支付宝），并自动切换到对应的配置模板。

### 2. 配置规则

#### 自动识别规则

上传文件后，系统会根据文件名自动识别账单类型：

- 包含"微信"、"wechat" → 自动使用微信配置
- 包含"支付宝"、"alipay" → 自动使用支付宝配置

#### 手动切换配置

您也可以通过配置区域的按钮手动切换：
- 点击"支付宝"按钮 → 使用支付宝配置模板
- 点击"微信"按钮 → 使用微信配置模板

#### 自定义规则

配置文件使用YAML格式，主要配置项：

```yaml
# 默认账户
defaultMinusAccount: 负方账户（通常是信用卡或现金）
defaultPlusAccount: 正方账户（通常是临时待分配账户）
defaultCurrency: 货币类型

# 微信规则
wechat:
  rules:
    - peer: 商户名称
      targetAccount: 目标账户
    - type: 交易类型
      targetAccount: 目标账户
    - method: 支付方式
      methodAccount: 支付账户

# 支付宝规则
alipay:
  rules:
    - peer: 商户名称
      targetAccount: 目标账户
    - method: 支付方式
      methodAccount: 支付账户
```

#### 规则匹配优先级

1. Peer（交易对方）匹配
2. Type（交易类型）+ Item（商品）匹配
3. Method（支付方式）匹配

#### 完整配置示例

参考 `config-example-full.yaml` 文件，包含详细的配置说明和示例。

### 3. 转换和下载

点击"开始转换"按钮，转换完成后可以：
- 查看转换摘要（成功记录数、跳过记录数等）
- 预览Beancount格式内容
- 下载.bean文件

## 账单导出指南

### 支付宝

1. 打开支付宝APP
2. 进入"我的" → "账单"
3. 点击"账单明细"
4. 选择时间范围
5. 点击"导出" → 选择CSV或Excel格式

### 微信

1. 打开微信APP
2. 进入"我" → "服务" → "钱包" → "账单"
3. 点击"常见问题" → "下载账单"
4. 选择用途："用于个人对账"或"用做证明材料"
5. 选择时间范围
6. 选择CSV或Excel格式
7. 填写邮箱并提交

## 常见问题

### Q: 为什么显示"文件为空或格式不正确"？

A: 可能的原因：
- 文件确实为空或只有表头没有数据
- 文件不是从官方渠道导出的
- 文件被Excel等工具修改过

### Q: 微信账单使用了支付宝的规则怎么办？

A: 系统会自动识别账单类型，如果识别错误，可以手动点击配置区域的"微信"按钮切换到微信配置。

### Q: 如何添加自定义规则？

A: 在配置文件中添加规则，例如：

```yaml
wechat:
  rules:
    - peer: 您的商户名称
      targetAccount: Expenses:Your:Category
```

### Q: 支持哪些年份的账单？

A: 支持任意年份的账单，包括2024年、2025年、2026年等。

### Q: 转换后的账户名称不对怎么办？

A: 在配置文件中修改 `defaultPlusAccount` 和 `defaultMinusAccount` 为您的实际账户名称。

## 技术支持

如果遇到问题，请检查：
1. 文件格式是否正确（CSV/XLSX）
2. 文件是否是从官方渠道导出的
3. 配置文件格式是否正确
4. 浏览器控制台是否有错误信息
